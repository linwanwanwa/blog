# Linwan's Blog

个人博客网站，基于 Vue 3 + Vite 构建。

## 技术栈

- Vue 3
- Vue Router
- Vite

## 运行项目

```bash
cd vue-blog
npm install
npm run dev
```

## 项目结构

```
vue-blog/
├── src/
│   ├── components/    # 组件
│   ├── views/        # 页面
│   ├── router/       # 路由
│   ├── stores/      # 状态管理
│   └── assets/      # 静态资源
└── public/          # 公共资源
```

## 特性

- 响应式设计
- 深浅色主题切换（根据时间自动切换）
- 毛玻璃效果
- iOS 风格背景
